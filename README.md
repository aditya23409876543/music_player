## ðŸš€ Getting Started

### Prerequisites
*   **Java Development Kit (JDK)** installed.
*   **Python 3.x** (for the local server).
*   A modern web browser (Chrome, Edge, Firefox).

### Installation & Run Steps

1.  **Clone/Download** this repository.
2.  **Add Music:** Place your `.mp3` files in the `web/music/` folder.
3.  **Compile & Generate Playlist:**
    Open a terminal in the project root and run:
    ```bash
    javac Song.java MusicPlayer.java PlaylistExporter.java
    java PlaylistExporter
    ```
    *This creates the `playlist.json` file used by the frontend.*

4.  **Launch the Player:**
    Double-click `start-server.bat`
    *OR run manually:*
    ```bash
    cd web
    python -m http.server 8000
    ```

5.  **Enjoy:** Open your browser to `http://localhost:8000`.
